## Dependencies

- Node >= 10.0
- npm >= 6.0

## Stack

- [React](reactjs.org)
- [xState](https://xstate.js.org/docs/)
- [CSS Modules](https://github.com/css-modules/css-modules)

## Setup

```
yarn install
```

## Running the application

```
yarn start
```
